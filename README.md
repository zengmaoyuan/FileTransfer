# FileTransfer
~~Too lazy to write an English README~~ \
FileTransfer 是一个轻量级的跨平台局域网（LAN）数据传输工具，基于 **TCP 协议**，它支持在同一网络下的设备之间快速、安全地传输**剪切板文本**与**任意类型的文件**。

## 核心特性

*   **局域网直连**：数据在局域网内直接点对点传输。
*   **TCP 协议保障**：采用 TCP 传输控制协议，确保传输稳定性。
*   **同步剪切板**：支持一键发送设备的剪切板文本。~~有可能会增加直接传输剪切板内文件功能~~

## 要求
*   **硬件要求**: 两台在同一个局域网内的电脑（全平台支持）
*   **环境要求**: **.NET Core 9.0** 及以上
